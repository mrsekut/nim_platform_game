# Nim Platform Game

![sample](https://github.com/mrsekut/nim_platform_game/blob/master/image/sample.gif)

## Building

- `$ nimble build` or `$ nim -d:release c nim_platform_game`

## Run

- `$ ./nim_platform_game`

## How to Play

- jump: `space`
- right: `l`
- left: `h`
- refresh: `r`
- quit: `q`
